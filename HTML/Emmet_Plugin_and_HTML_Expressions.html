Working with Emmet's "Wrap with Abbrevation" function

Sometimes when writing HTML, you want to quickly wrap an element with an additonal element.
This can be a cumbersome process of formatting HTML, making sure you add the start and ending tags, etc.

Luckily, this is where the Emmet plugin in your IDE can help with auto formatting. Let's take the HTML ENCODED NOTATION OUTPUT
from above, I will use the Emmet plugin to automatically format a "Title" Wrapper to the paragraph Element.
This is done by putting your cursor inside the start tag you'd like wrap, running the custom keybinding asscoaietd with the desired action,
in this case the action "Wrap with Abbreviation" action which I have set to "Shift + CTRL + A" will open up the prompt box
where I can enter my wrapper defintion: .wrapper>h1{Title}+.content
