Working with Emmet's "Wrap with Abbrevation" function

HTML ENCODED NOTATION OUTPUT

<div id="page">
  <p>Hello, World!</p>
</div>

HTML EXPRESSION OUTPUT

Hello world

Sometimes when writing HTML, you may need to quickly wrap an element with an additional element. This process can be cumbersome, involving formatting HTML and ensuring you add the start and ending tags.

Luckily, the Emmet plugin in your IDE can simplify this with auto-formatting. Let's use the HTML ENCODED NOTATION OUTPUT from above. I'll use the Emmet plugin to automatically format a "Title" wrapper around the paragraph element.

To do this, place your cursor inside the start tag you'd like to wrap and run the custom keybinding associated with the desired action. In this case, the "Wrap with Abbreviation" action, set to "Shift + CTRL + A," will open a prompt box where you can enter your wrapper definition:

.wrapper>h1{Title}+.content.

In my IDE, I will click the "*" in the expression below, intiate my action using "Shift + CTRL + A", and finally type
my expression .wrapper>h1{Title}+.content to add and format my title wrapper

<div id="page">
  <p*>Hello, World!</p>
</div>

After following these steps, my code is now:

    <h1>Title</h1>
    <div class="content">

  </div>
